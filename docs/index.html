Project 0

Two Sum

A C++ implementation of the classic Two Sum problem using a hash map to find a valid pair in linear time.

  • Algorithm: complement lookup with unordered_map
  • Complexity: O(n) time, O(n) space
  • Testing: 23 cases across simple, edge, single-target, and stress tests
